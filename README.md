1) Файл R-CNN.ipynb содержит полный пайплайн для обучения модели R-CNN от импорта датасета до оценки параметров полученной модели
2) Файл data_preprocessing_kitti.ipynb содержит предобработку данных для обучения модели YOLOv8n на ultralytics
3) Файл train_base_yolo.ipynb содержит код для импорта модели YOLOv8n из библиотеки ultralytics, обучения модели на неаугментированных данных, а также для дообучения модели
4) Файл train_yolo_augmented.ipynb содержит код для обучения модели YOLOv8n на агментированных данных и со специально подобранным training config для достижения наибольшей точности
5) Файл evaluation_model.ipynb содержит код для точной оценки параметров всех обученных ранее моделей
